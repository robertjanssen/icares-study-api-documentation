# Example Library

The Example Library contains ready-to-use GraphQL examples for the Icares Study API.

The examples can be copied directly into the GraphiQL in-browser tool or adapted for use in your own application.

## Categories

### Mutations

GraphQL mutations used to create, start and submit study tests.

- [Mutations](Mutations/README.md)

### Queries

GraphQL queries used to retrieve form definitions, study results and shared registrations.

- [Queries](Queries/README.md)

## Typical API Workflow

1. Invite a study test.
2. Retrieve the test form.
3. Start the study test (optional).
4. Submit the completed answers.
5. Retrieve the results.
6. Retrieve shared registrations (optional).
