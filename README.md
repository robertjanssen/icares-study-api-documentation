# Icares Study API Developer Documentation

Welcome to the official developer documentation for the Icares Study API.

This repository contains everything required to integrate with the Icares Study API, including Quick Start Guides and ready-to-use GraphQL examples.

## Documentation

### Quick Start Guides

Start here if you are integrating with the API for the first time.

- Icares Study API Quick Start Guide
- GraphiQL In-Browser Tool Quick Start Guide

### Example Library

The Example Library contains ready-to-use GraphQL examples that can be copied directly into the GraphiQL in-browser tool or your own application.

Examples are grouped into:

- Queries
- Mutations

## Typical integration workflow

A typical integration consists of the following steps:

1. Create a test invitation.
2. Retrieve the test form.
3. Present the questions to the user.
4. Submit the completed test.
5. Retrieve the calculated results.

## Authentication

All requests use HTTP POST and are authenticated using a Bearer Token.

For more information, see the **Icares Study API Quick Start Guide**.
